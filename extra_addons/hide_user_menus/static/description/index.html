Frequently Asked Questions

Find answers to common questions about installation, configuration, and using the app. If you don't see your question here, feel free to reach out — we're happy to help!

What does the Hide User Menus app do?
It hides unnecessary menu items from the top right corner of the Odoo interface, such as Documentation, Support, Shortcuts, and others, providing a cleaner and distraction-free UI.
Which menu items are hidden by this app?
The app hides Documentation, Support, Shortcuts, Onboarding, and My Odoo.com account links from the user menu at the top right.
Is any configuration needed after installation?
No configuration is required. The module works immediately after installation with no setup or code changes needed.
Will it affect admin or only regular users?
By default, it affects all users. If role-based visibility is required, a custom version can be provided.
Does this module impact any other menus or functionality?
No. It only modifies the visibility of items in the top-right user dropdown. Other menus and Odoo features remain unaffected.
Can I choose which menu items to hide?
The default version hides all supported items. For selective hiding, contact us to request a customizable build.
Is it compatible with Odoo Enterprise?
Yes, the module is compatible with both Odoo Community and Enterprise editions.
